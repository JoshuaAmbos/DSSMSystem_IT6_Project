# Ukay-Ukay Sales & Stock Tracker

A streamlined management tool designed to simplify the daily operations of a thrift business. This system moves your inventory from messy notebooks to a clean digital dashboard.
Core Functionality

    **Inventory Tracking**: Monitor stock levels from "available" to "sold."

    **Sales Logging**: Record daily transactions and track revenue.

    **Profit Overview**: Automatically calculate earnings by subtracting bale costs from total sales.

    **Simple Reporting**: View daily summaries to see which items are moving the fastest.
